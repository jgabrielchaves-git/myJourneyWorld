# 00 — Project Charter

## Nome do Projeto

Journey World

## Missão

Gamificar o aprendizado infantil através de mundos independentes.

## Problema

Manter uma criança motivada a praticar piano diariamente, de forma consistente, sem que a prática vire uma obrigação desagradável.

## Hipótese do MVP

> Uma jornada gamificada aumenta a motivação e a consistência de uma criança nos estudos de piano.

O MVP existe unicamente para validar essa hipótese. Nenhuma funcionalidade deve ser adicionada se não contribuir diretamente para testá-la.

## Primeiro Mundo

Piano — Mundo Piano.

## Público-Alvo

Criança (usuária principal) + pais/responsáveis (supervisão via Painel dos Pais).

## Critério de Sucesso do MVP

A criança volta espontaneamente, todos os dias, para estudar piano dentro da experiência.

O MVP deve transmitir conforto e vontade de voltar todos os dias.

## Visão de Longo Prazo

Após validar o Mundo Piano, expandir para novos mundos (Ginástica Rítmica, Escola, Família, Virtudes, Jornada Espiritual), todos compartilhando o mesmo personagem, casa, economia, loja, álbum e progressão. Ver [Product_Vision.md](Product_Vision.md).

## Restrições

- Stack limitada a HTML, CSS e JavaScript durante o MVP.
- Persistência via LocalStorage, preparada para futura migração online.
- Decisões de gameplay, identidade visual, direção de arte, economia e game design **não** são tomadas por Claude — vêm da documentação/produto. Ver [CLAUDE.md](../CLAUDE.md).
